# Helm

- helm install mysql .

- helm install backend .

- helm install frontend .

- helm list

- helm upgrade mysql → we run this when there is any changes in mysql

- helm status mysql

- helm history mysql

- helm rollback mysql 2 → when you upgraded new version of application and having some errors, then use this command for get specific previous versions

- helm uninstall frontend .

- helm uninstall backend .

- helm uninstall mysql .
